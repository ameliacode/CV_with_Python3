### Getting Started

시작하기 전 라이브러리를 설치해주세요 (자세한 내용은 각 단원별 노트북 참고)

```
pip install Pillow
pip install scikit-image
pip install sklearn
pip install tensorflow
pip install opencv-contrib-python
```

### Dependencies
Python 3.7  
Opencv 4.4.0  
Tensorflow 2.3.1 
Pillow 8.0.1  
Scikit-image 0.18.1  
Scikit-learn 0.23.2
